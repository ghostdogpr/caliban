# About

Caliban is a project developed by Pierre Ricadat aka @ghostdogpr (opens new window).

The name is inspired by the SF novel and tv series The Expanse (opens new window).

Thanks:

  • @deckerst (opens new window) for the logo
  • @yoohaemin (opens new window), @adamgfraser (opens new window) and @jdegoes (opens new window) for the early feedback
  • the Morpheus (opens new window) library in Haskell for the inspiration leading to the GraphQL backend implementation
  • the elm-graphql (opens new window) library for the inspiration leading to the GraphQL client implementation
  • all Caliban's contributors (opens new window)!
